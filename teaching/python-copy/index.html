UBC + LTS Python Coding Club

Together with a group of passionate graduate student volunteers, I have been organizing, creating lessons for, and teaching an afterschool Python course for middle- and high-school aged girls.

Starting from no programming knowledge, we introduce the students to the fundamentals of programming using Trinket, a free-to-use online-hosted Python and Blockly editor. Over the course of 6 weeks, students learn everything from variable creation to while loops, and top off the program by creating their own interactive game.

Check out the website here for more information/inspiration, or check out our ever-developing lesson plan if you want to run something similar!
